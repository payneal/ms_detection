# Main one used: MNI152_T1_1mm.nii.gz

  * MNI152_T1_1mm.nii.gz is a standardized 3D brain template used as a common coordinate system in neuroimaging.

## Breaking down the name:

* MNI → Montreal Neurological Institute
* 152 → constructed by averaging 152 healthy adult brains
* T1 → T1-weighted MRI contrast
*  1mm → isotropic voxel resolution (1×1×1 mm)
* .nii.gz → compressed NIfTI image format
